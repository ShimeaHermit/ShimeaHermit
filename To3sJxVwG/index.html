给网站增加了可互动的看板娘

2023-03-30

看板娘仅在pc下可见

基于up主“有云转晴”的教学制作的
代码如下，插入indexl.html的<head><head>之间即可
<!--看板娘 -自定义大小，隐藏对话框和对话框高度-->
<style type="text/css">
#waifu #live2d {
height: 300px!important;
width: 300px!important;
}
#waifu-tips {
top: -60px;
/display:none !important; #隐藏对话框（删除/即可隐藏对话）/
}</style>
<!--看板娘加载指定模型-->

<script>
localStorage.setItem('modelId', '24'); /选择角色/
localStorage.setItem('modelTexturesId', '1');/选择衣服（皮肤）/
/插入后F12查看元素即可查看对应角色和皮肤的代号/
</script>

<!--配置看板娘左右位置-->

<script src="https://api.itggg.cn/live2dnew/left/index.js"></script>
